# microblog4

PythonフレームワークのDjangoを使用したTwitterのクローンアプリです。

機能

・記事一覧表示
・記事詳細
・記事投稿１４０文字（＊）
・記事更新（＊）
・記事削除（＊）
・ログイン・ログアウト
・ページネーション
（＊）はログイン時のみ操作可能です。ログイン用のアカウント情報をのせておきます。
